## Step 3 — Cover page formatting (manual)

The UPT manual specifies:
- University name, faculty, department: Times New Roman, **14pt**, BOLD CAPITALS
- Thesis title: Times New Roman, **16pt**, BOLD CAPITALS
- "TIRANË, MUAJI 2026": Times New Roman, **12pt**, CAPITALS

Manually adjust the font sizes on the first page (Heading 1 elements) to match.

You may also want to:
- Add the UPT logo at the top (download from upt.al, paste, centre, resize to ~3 cm width)
- Add white space so the title sits roughly in the upper-middle of the page
- Add a horizontal rule above your name and supervisor name

## Step 4 — Update the Table of Contents

The TOC was auto-generated by Pandoc. After editing, refresh it:

1. Click anywhere in the TOC.
2. Right-click → Update Field → Update entire table.
3. Word renumbers all entries to match your edits.

## Step 5 — Add List of Tables and List of Figures

After the Table of Contents, insert:

**List of Tables:**
1. Click where you want the list.
2. References → Insert Table of Figures.
3. Set "Caption label" to "Table".

**List of Figures:**
1. References → Insert Table of Figures.
2. Set "Caption label" to "Figure".

For these to populate, you must caption your tables and figures using Word's References → Insert Caption (with label "Table" or "Figure"). Currently the tables in the docx don't have proper Word captions; they have inline bold labels like "Table 4.1.". You'll want to convert these:

1. Select the bold "Table 4.1." text above each table.
2. References → Insert Caption → Label: "Table" → OK.
3. Word inserts a proper caption that the auto list can pick up.
4. Repeat for figures.

This is tedious but it's the only way Word's auto-list-of-tables works.

## Step 7 — Page numbering

The UPT manual specifies:
- Preliminary pages (Abstract, TOC, etc.): Roman numerals (i, ii, iii, ...)
- Main text from KREU I onward: Arabic numerals (1, 2, 3, ...)
- Page numbers at bottom centre

In Word:
1. Insert → Page Number → Bottom of Page → Plain Number 2 (centred).
2. Section break before KREU I (Layout → Breaks → Section Break: Next Page).
3. In the new section: Insert → Page Number → Format Page Numbers → Start at 1, Number format = 1, 2, 3.
4. In the prelim section: Format Page Numbers → Number format = i, ii, iii.

## Step 8 — Header

Per UPT manual: **no header**. Footer is only for page numbers. Already correct.

## Step 9 — KREU titles on new pages

Each KREU should start on a new page. Pandoc has done this for KREU I and Abstract. Verify each KREU II, III, IV, V starts on its own page; if not, place the cursor before the KREU title and press Cmd+Enter (Mac) or Ctrl+Enter (Win) to insert a page break.

## Step 10 — Thesis length check

UPT requires Master thesis to have **at least 50 pages and 17,000 words**, excluding preliminary pages, references, lists, and appendices.

To check:
- Word: Tools → Word Count, or Cmd+Option+W.
- Current draft: ~16,000 words. If short of 17,000, expand the discussion sections in KREU IV (especially the "combined findings" and per-class breakdown discussion) and the future-work section in KREU V.

## Step 11 — Print-ready check

Before printing:
- File → Print → Preview each page; ensure no orphan headings or split tables.
- Margins: File → Page Setup → confirm 2.54 cm top/bottom/right and 3.8 cm left.
- Font: Edit → Find → use the "Replace formatting" feature to confirm everything is Times New Roman 12pt.

## Step 12 — Bind copy (UPT requirement)

Per UPT manual:
- Master thesis: "model book" binding, with first and last pages on cardstock (karton).
- Cover must show: University name, Faculty, Department (in CAPS), thesis title, year, with the formatting specified above.
- Include a CD with the .docx and .pdf in a sleeve glued to the last page.

Find a printer in Tirana that does academic thesis binding (most copy shops near the university will know the UPT format).
